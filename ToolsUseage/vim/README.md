&nbsp;&nbsp;**基本操作命令：**
```
yy ==> 复制一行
dd ==> 删除一行
dw ==> 删除单词
:et nu ==> 设置行号
:et nonu ==> 取消设置行号
:{number} ==> 跳转行数
/{string} ==>  匹配字符串
:w ==> 保存
:q ==> 退出
:q! ==> 退出不保存
:wq ==> 退出并保存
ESC ==> 模式切换(插入模式|命令模式|可视模式)
u ==> 撤销上一步
i ==> 插入
o ==> 插入下一行
shift + o ==> 插入上一行
h ==> 左
j ==> 下
k ==> 上
l ==> 右
shift + g ==> 最后一行
gg ==> 开头行

```

&nbsp;&nbsp;**模式切换：**
```
i ==> 插入模式
shift + v: 可视模式
:xxx | /xxx ==> 命令模式 
```

&nbsp;&nbsp;**一些骚操作：**
```
#多开文件
:split ==> 横着开
:vsplit ==> 竖着开

```
