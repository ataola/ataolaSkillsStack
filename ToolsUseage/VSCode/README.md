            "defines": [
                "_DEBUG",
                "UNICODE",
                "_UNICODE"

**&nbsp;&nbsp;代码模板的新建:**

>File ==> Preferences ==> User Snippets

&nbsp;&nbsp;&nbsp;&nbsp;在VSCode菜单栏选择楼上那位的路径，然后这里以HTML为例，配置的JSON文件如下

```
"HTML template": {
"prefix": "html5",
"body": [
"<!DOCTYPE html>",
"<html lang=\"zh\">",
"<head>",
"\t<meta charset=\"UTF-8\">",
"\t<title>${1:Page Title}</title>",
"</head>",
"<body>",
"\t$0",
"</body>",
"</html>"
],
"description": "Simple HTML5 starting point"
}
```
