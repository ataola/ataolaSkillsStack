## 牛客取经

**与卿可津**

首位可以零，或者首位不能为零，我看很多人都没考虑首位不能为零的问题。

```js
function isUSD(str) {
    var re = /^\$([1-9]\d{0,2}(,\d{3})*|0)(\.\d{2})?$/;
    return re.test(str);
}
```
