## 牛客取经

**希留**

本题需要注意格式，开头^和结尾$必须加上来限定字符串，3个数可表示为\d{3}，4个数则为\d{4}，{n}表示前面内容出现的次数。正则表达式可写作/^\d{3}-\d{3}-\d{4}$/，有相同部分\d{3}-，因此也可写作/^(\d{3}-){2}\d{4}$/

```js
function matchesPattern(str) {
    return/^(\d{3}-){2}\d{4}$/.test(str);
}
```
